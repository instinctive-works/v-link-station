# 概要 / About V-Link Station
V-Link Station は、モーションキャプチャおよび映像データを一括で受信・収録し、再送出するためのミドルウェアサーバーです。複数のデバイスやソースからの入力を統合し、**収録の一括操作（一斉レコーディング）**を可能にすることで、各種XRワークフローを支援します。

V-Link Station is a middleware server designed to aggregate, record, and re-broadcast motion and video data from multiple sources in real-time. It streamlines various XR workflows by providing centralized control for batch recording, enabling users to manage and trigger multiple capture devices simultaneously.
